== Description ==

**Kamahra Unique Words** is a lightweight plugin that takes user-inputted text, removes duplicates, and outputs a list of unique words using a separator you choose (comma, newline, etc).

Includes:
- Live word processing
- Customizable separator
- Copy-to-clipboard button
- Soft limit warning for large input
- Admin color customization

== Installation ==

1. Upload the plugin ZIP via `Plugins → Add New → Upload Plugin`
2. Activate the plugin through the "Plugins" menu
3. Add shortcode `[kamahra_unique_words]` to any page/post
4. Customize UI colors under `Settings → Kamahra Colors`

== Frequently Asked Questions ==

= Does it support non-English words? =
Yes, as long as they are standard UTF-8 characters.

= Can I change the default separator? =
Yes — just select the separator provided.
